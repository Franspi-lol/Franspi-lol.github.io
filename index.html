Proyectos

Biblioteca de Peliculas

Proyecto sobre un sistema basico de administracion de peliculas para usuarios realizado en C

Se utilizaron estructuras compuestas, punteros, recursividad y almacenamiento en archivos

Biblioteca de Videojuegos

Proyecto sobre un sistema basico de administracion de videojuegos para usuarios a partir de un archivo json ya formateado con un listado de juegos. Realizado en Java

Se utilizaron conocimientos en programacion orientada a objetos, genericidad, interfaces, excepciones, consumo de archivos JSON y utilizacion de archivos para guardar datos de usuarios

Batallas Pokemon

Aplicacion Angular sobre un sistema capaz de registrar usuarios y permitirles jugar a un Videojuego de combate por turnos basado en el videojuego Pokemon. La app permite al usuario jugar con otra persona, revisar historial y observar datos de los Pokemons obtenidos a partir de la PokeApi

Se utilizo Angular, Formularios reactivos, Consumo de APIs, JSON-Server y CSS
